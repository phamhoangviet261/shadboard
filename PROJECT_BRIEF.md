# Lensora Project Brief

## One-liner

Lensora is an AI-powered eyewear ecommerce store and CMS for managing glasses products, collections, product content, and SEO.

## Goal

Build a realistic ecommerce frontend project that helps demonstrate strong frontend engineering skills:

- Ecommerce UI
- Product listing
- Product detail UX
- Advanced filters
- Admin CMS
- Product forms
- Image upload
- Data table
- AI-assisted content generation
- Responsive design
- Clean architecture

## Target User

1. Customers browsing glasses
2. Store admins managing products
3. Recruiters/developers reviewing the implementation quality

## Brand Feel

Lensora should feel premium, minimal, clean, modern, and trustworthy.

Inspired by:

- Gentle Monster
- Apple product pages
- Linear dashboard polish
- Modern boutique ecommerce

## MVP Scope

Public:

- Collection page
- Product grid with filters
- Product detail page
- Product quick view

Admin:

- Product list table
- Product create/edit form
- Collection management
- Image upload preview
- AI writer for product descriptions and SEO

## Success Criteria

The project should look like a real production ecommerce app, not a simple portfolio demo.
